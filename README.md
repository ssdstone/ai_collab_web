# AI 合作介绍网页生成器

输入甲乙双方信息，由 DeepSeek AI 自动生成**风格适配行业**的专业合作介绍 HTML 网页。

## 快速开始（Windows 11）

### 方式一：双击启动（推荐）
直接双击 `启动.bat`，脚本自动安装依赖并打开浏览器。

### 方式二：手动启动
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```
然后打开浏览器访问 `http://127.0.0.1:5000`

---

## 使用说明

1. 在页面填写**甲方公司名称**和**AI 功能描述**
2. 填写**乙方公司名称**（越具体越好）
3. 输入你的 **DeepSeek API Key**（`sk-xxx` 格式）
4. 点击「生成合作介绍网页」
5. 等待 15～30 秒，页面自动展示结果

**输出方式：**
- 应用内 iframe 实时预览
- 「新标签预览」：在新窗口查看完整效果
- 「下载 HTML」：保存到本地，可直接发给客户

## 风格自动适配规则

| 乙方行业 | 自动风格 |
|---------|---------|
| 律师/会计/金融事务所 | 深色商务风：深蓝+金色，Serif 字体 |
| 科技/互联网/AI 公司 | 科技暗色风：霓虹渐变，几何动效 |
| 医疗/健康/制药 | 清洁医疗风：白色+浅蓝/绿，圆润卡片 |
| 教育/培训机构 | 活力教育风：明亮配色，友好字体 |
| 制造业/工业 | 工业硬核风：深灰+橙色，粗体 |
| 消费品/零售/时尚 | 现代商业风：简洁时尚，大图 |
| 其他 | 通用专业商务风 |

## 项目结构

```
ai_collab_web/
├── app.py              # Flask 主程序
├── requirements.txt    # 依赖列表
├── 启动.bat            # Windows 一键启动
├── templates/
│   └── index.html      # 主界面
└── output/             # 生成的 HTML 文件（自动创建）
```

## 环境要求

- Python 3.9+
- 网络可访问 DeepSeek API (`api.deepseek.com`)
- DeepSeek API Key（[申请地址](https://platform.deepseek.com)）
